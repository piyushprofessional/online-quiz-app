# Online Quiz App

A responsive quiz project made with HTML, CSS and JavaScript.

## Features

- Login with name and optional email
- Category selection
- Select 5, 10 or all questions
- Timer-based quiz
- Previous / Next questions
- Question map
- Auto-submit when time ends
- Score calculation
- Grade result
- Review correct and wrong answers
- Leaderboard using localStorage
- Admin panel to add, edit and delete questions
- Reset default questions
- Dark mode
- Mobile, tablet and laptop responsive design

## How to run

Open `index.html` directly in your browser.

No backend and no installation required.

## How to deploy on Netlify

1. Extract the ZIP file.
2. Go to Netlify.
3. Add new project.
4. Deploy manually.
5. Drag and drop the complete extracted folder.
